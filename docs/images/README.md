# Images Directory

Place your project screenshots and images here.

## Usage in Markdown

Reference images using standard Markdown syntax:

```markdown
![Project Screenshot](images/project-screenshot.png)
![Project Logo](images/project-logo.jpg)
```

## Recommended Structure

- `images/projects/` - Project screenshots and images
- `images/about/` - Personal photos or team photos
- `images/logos/` - Technology logos or company logos

## Image Guidelines

- Use descriptive filenames (e.g., `ecommerce-dashboard.png`)
- Optimize images for web (recommended formats: PNG, JPG, WebP)
- Recommended max dimensions: 1920x1080 for screenshots
- Keep file sizes under 500KB when possible
